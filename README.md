🎵 Amit Music

Amit Music is a modern, Spotify-inspired web music player that lets users create and manage YouTube-based playlists in a simple and responsive interface.

✨ Features

- 🎵 Add songs using YouTube links
- ▶️ Built-in YouTube player
- 📂 Playlist management
- 🔍 Search songs
- ⏮️ Previous / ⏯️ Play / ⏭️ Next controls
- 💾 Local Storage support (playlist is saved automatically)
- 🌙 Dark theme UI
- 📱 Mobile-friendly responsive design

📁 Project Structure

Amit-Music/
│── index.html
│── style.css
│── script.js
│── README.md

🚀 How to Run

1. Download or clone the project.
2. Open the project folder.
3. Double-click "index.html" or open it in your web browser.
4. Paste a YouTube video link.
5. Click Add Song and enjoy your playlist.

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- YouTube Embed Player
- Local Storage
